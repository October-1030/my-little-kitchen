## 📝 使用说明

### 添加新菜谱

1. 拍照或找到菜品照片
2. 发给 MIMI："帮我添加菜谱"
3. MIMI 会问你：
   - 需要 AI 生成精美版吗？
   - 菜名是什么？
   - 分类（早餐/午餐/晚餐/小食）
   - 评分、食材、做法等
4. 自动整理到对应文件夹

### 查看菜谱

- 打开对应分类文件夹
- 每道菜一个 `.md` 文件
- 可以用任何 Markdown 编辑器查看（VS Code、Typora 等）

### 搜索菜谱

- 用文件搜索功能
- 或者在 README.md 中按标签查找
